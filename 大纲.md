# 【崩坏指挥终端 - 独立里世界UI系统设计大纲】

## 一、 双轨驱动资源体系
为了让科技解锁与战术施放互不干涉，建立健康的经济内循环。

**1. 【崩坏能研究点】 (Honkai Research Points)**
- 定位：仅用于在UI面板中解锁【文明科技树】与【领袖技能树】的节点。
- 产出逻辑：每解锁 1 个原版科技/市政，每回合产出 +1。三大组织建筑提供少量加成。

**2. 【崩坏能】 (Honkai Energy)**
- 定位：动态运转的战术能量池。用于维持律者权能【光环】和释放【大招】。
- 产出逻辑（随时代膨胀制）：
  * **疆域共鸣 (随时代进化)**：崩坏随着文明的发展而变强。帝国疆域内的每一个单元格均产出崩坏能，且**单格产出随时代系数暴增**（如：远古1.0 -> 古典1.2 -> 中世纪1.5 -> 工业3.0 -> 信息8.0）。用庞大的领土和时代乘区，直接碾压后期的资源需求！
  * **核心枢纽网**：三大组织总部（天命、逆熵、世界蛇）及遍布帝国的附属设施（天命支部、盐湖基地、海渊城、世界蛇分部等），不仅充当前置条件，更是巨型“崩坏能发电机”，提供极高额的固定回合产出。
  * **以战养战**：击杀敌方单位汲取额外的崩坏能。
  * **电池被动**：持有律者核心/神之键充当“电池”提供高额回蓝。
- 数值平衡哲学：对冲【崩坏灾难】。玩家在原版经济（科文金）上面临越来越严苛的时代Debuff，但在里世界会获得呈指数级增长的【崩坏能】，玩家必须利用这些崩坏能在UI中购买降维打击级别的超级Buff来破局。

## 二、 UI 终端三页架构

<details>
<summary> 点击展开：三页架构详情 </summary>

**【Page 1：崩坏源流科技树 (1主+3支线)】** 
- 适用范围：所有崩坏帝国领袖共享的科技骨架。
- 终点神座：每当一条线被点到【最终级别节点】，发放一个【权能解锁许可】兑换池中的律者权能。

**【Page 2：虚数共鸣终端 (跨领袖联动网络)】** 
- 适用范围：动态读取对局中存在的所有“崩坏帝国文明领袖”。
- 核心内容：与其他崩坏领袖进行羁绊连线，单向解锁微弱加成，双向解锁瞬间触发超级联动战力。

**【Page 3：领袖技能树 & 权能战术面板】** 
- 适用范围：领袖独特的发育树及律者核心“操控台”。
- 核心内容：切换光环耗蓝（维持跃迁等）、瞬间大招释放（接管野蛮人等）。律者本尊在此直接1折购买专属权能。
</details>

## 三、 兵工生产线与世代更迭
- **保留原版种田**：UI 只负责发放蓝图。女武神和泰坦依然需要在城市里花费【生产力(齿轮)】和【隐藏战略资源】建造。
- **数值克制 (世代淘汰)**：比如“崩坏炉过载”科技仅为【Mk1】+2力。升级到【Mk2】时该加成作废，确保小科技叠再多也不会让大后期数据爆炸。

## 四、 巅峰权能体系：【十大律者权能】
这是一个极具深度的“硬件体系”。解锁一颗核心包含：
1. **电池被动**：永久提供 +20 崩坏能/回合。
2. **奇观图纸**：如解锁奇观【虚数之门】。
3. **光环状态**：如开启后 -100能/回合，全军获得全图跳跃。
4. **瞬间大招**：如瞬间消耗 1500能，全图接管野蛮人。

*(注：其余权能具体组合待设计)*

## 五、 UI 科技树节点详设：【中央主线 (崩坏源流)】
主线科技是帝国的基石，采用**网状多线结构**。分为【能源工程线】（基建与资源）、【生物圣痕线】（人口与适应）、【虚数量子线】（空间与尖端理论）。必须通过它们来解锁隐藏战略资源和支线准入权限。

### 【第一世代：远古-古典（启蒙与适应）】

## 九、 网状画板连线视觉排版构思
整个画布被切分为四条明显的垂直轨道（Track）：
1. **极左轨【天命区】**：女武神与信仰的双线交织。
2. **中左轨【主线区】**：能源与生物交叉的核心脉络。
3. **中右轨【逆熵区】**：机甲与重工业的双线交织。
4. **极右轨【世界蛇】**：深潜于黑暗中的独立情报网络。
每一行代表一个“时代层”，节点之间互相存在复杂的 X 型、Y 型前置连线！
